# 更新记录

## v.0.10.0 - 2026-07-02

发布分支：`v.10.0`

### 文库监测

- 将文库监测升级为独立监测看板，整体交互和视觉靠近网盘监测。
- 增加免登录优先文库来源识别和默认扫描来源，包括人人文库、金锄头文库、MBA 智库文档、360 文库、腾讯文库、夸克文档、淘豆网、文档之家、搜弘文库等。
- 登录限制较重的平台不作为默认高频来源，避免绕登录、验证码或付费下载。
- 调整文库扫描默认深度，修复文库命中列表被网盘结果挤占的问题。
- 文库详情页改为网盘监测详情风格，并移除中间文件清单区域。

### 网盘与文档镜像

- 网盘监测和文库监测新增镜像文件保留能力。
- 网盘命中支持 HTML 镜像快照和旧数据缺失镜像时的回填。
- 文库命中支持公开 HTML、截图、快照路径等镜像线索保留。
- 详情页补充“镜像文件”入口，便于复核当时页面证据。

### 登录与账号

- `admin` 默认密码调整为 `123456`。
- 支持本地密码文件初始化，避免启动时因 `DARKWEB_AUTH_PASSWORD` 未配置导致无法登录。
- 增加修改密码接口和前端入口，后续可在系统内变更默认密码。

### 版本与发布

- 增加 `version.json`，版本标记为 `v.0.10.0`，更新分支为 `v.10.0`。
- 后端 API 版本信息更新为 `v.0.10.0`。
- 前端包版本更新为 `0.10.0`。
- 新增版本检查能力，用于读取本地版本并比对 GitHub 分支。
